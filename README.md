# ClinicalMind

Professional AI-powered medical platform for healthcare institutions.

## Structure
- `frontend/`: Next.js web application
- `backend/`: FastAPI Python server
- `ml_models/`: AI/ML model artifacts and prediction logic
- `datasets/`: Data storage for training and validation
- `docs/`: Documentation and architecture designs
- `docker/`: Containerization and orchestration
- `tests/`: Testing suite
- `scripts/`: Automation and maintenance scripts
